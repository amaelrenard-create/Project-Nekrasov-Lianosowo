## Название проекта: 
Анализ корпуса поэзии объединения "Лианозовская школа"

## Описание:
В корпус вошли произведения трех поэтов: Эдуарда Лимонова, Всеволода Некрасова и Генриха Сапгира. Произведения охватывают пласт времени с 1966 года до 1991.

## Структура Проекта:
```
corpus_project/
├── corpus/
│   └── *.txt                  # Тексты стихотворений
│
├── data/
│   └── metadata.csv           # Таблица с метаданными произведений
│
├── results/
│   ├── report.txt             # Текстовый отчёт со статистическими выводами
│   └── statistics.csv         # Статистика по каждому произведению
│
├── main.py                    # Основной исполняемый файл проекта
├── file_utils.py              # Модуль для работы с файлами
├── text_utils.py              # Модуль для анализа текста
```

## Как Запустить
```
python main.py
