## Название проекта: 
Анализ корпуса поэзии объединения "Лианозовская школа"

## Описание:
В корпус вошли произведения трех поэтов: Эдуарда Лимонова, Всеволода Некрасова и Генриха Сапгира. Произведения охватывают пласт времени с 1966 года до 1991.

## Структура Проекта
corpus/ - тексты стихотворений в формате .txt
data/metadata.csv - таблица с метаданными ( ).
results/ - папка с результатами анализа
results/report.txt - текстовый файл с описанием всех статистических данных
main.py - основной исполняемый файл проекта
results/statistics.csv - статистика по каждому произведению
main.py - основной исполняемый файл проекта.
file_utils.py - модуль с функциями для работы с файлами.
file_utils.py - модуль с функциями для анализа текста

## Как Запустить
python main.py
